Tag gMapControl

This component adds the control to the ice:gMap. This component must need to be a child of ice:gMap component. (e.g.)
<ice:gMap>
<ice:gMapControl name="GLargeMapControl"/>
<ice:gMapControl name="GScaleControl"/>
</ice:gMap>
