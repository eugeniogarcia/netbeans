Tag rowSelector

The Row Selector tag enables row selection for DataTables. The Row Selector has the following attributes
  • clickedRow The row index of the last (de)selected row. Settable ValueBinding.
  • multiple When true, more then one row can be selected. Default is false.
  • mouseOverClass The CSS style to apply when the mouse is over an unselected row.
  • selectedClass The CSS style to apply when a row is selected.
  • selectedMouseOverClass The CSS style to apply when the mouse is over a selected row.
  • selectionAction The Action method to call when a row is clicked. Done after the selectionListener.
  • selectionListener The listener method to call when a row is clicked.
  • toggleOnClick Do row selection on mouse click. Default is true.
  • toggleOnInput Do row selection, even when interacting with input field in table cell. Default is true.
  • value The boolean value that indicates this row is selected. This value should be defined in the 'var' of the dataTable. Required.
